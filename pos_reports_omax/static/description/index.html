POS Reports

Overview

This module helps to print POS Session Z Report, POS Receipt with Logo & Barcode, POS day wise product sales report in PDF and Excel, POS Tax Receipt, POS Session Profit & Loss Report(POS Margin Report) and POS Session Report with the summary of Session Orders, Session Payments, Session Pickings, Session Invoices and Barcode.

Feature

  • POS Session Z Report
  • POS Receipt with Logo & Barcode
  • POS Weekdays Sales Report in PDF and Excel
  • POS Payment Report in PDF and Excel
  • POS Tax Receipt
  • POS Session Profit & Loss Report(POS Margin Report)
  • POS Session Report with the summary of Session Orders, Session Payments, Session Pickings, Session Invoices and Barcode

How It Works


POS Session Z Report


Session form view : User can print session report from session form view [Button and print menu].



Session list view : User can print any numbers of session report from list view.



POS Config : Configuration for allow to print Session Z Report directly from POS Screen.



POS Screen : From here user can print Session Z Report.
