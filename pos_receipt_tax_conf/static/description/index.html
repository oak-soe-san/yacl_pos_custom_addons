Features

  • This module helps to showing the tax details with various cases.

POS Receipt Tax Configuration

  • User can configure the POS receipt tax details from the setting of the shop.

Default

  • By selecting the default option, It works regular without any changes.

Show All Taxes

  • By selecting the Show All Taxes option, It shows all taxes detail with various type of taxes only it doesn't show Total Amount of Taxes.


Only Total Amount of Taxes

  • By selecting the Only Total Amount option, It shows only the Total Amount of all Taxes.
  • Note: Total Taxes only show if applied taxes is inclusive(included).


Hide Taxes Detail

  • By selecting the Hide Taxes Detail option, It hides the tax details from the receipt.
